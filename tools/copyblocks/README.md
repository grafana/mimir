# copyblocks

> [!WARNING]
> **Deprecated:** the standalone `copyblocks` tool is deprecated and will be removed in a future release.
> Use [`mimirtool blocks copy`](../../cmd/mimirtool) instead.

This program can copy Mimir blocks between two buckets or from a bucket to a backfill destination. Its documentation has moved to [`docs/internal/tools/copyblocks.md`](../../docs/internal/tools/copyblocks.md).
