# Split Blocks

> [!WARNING]
> **Deprecated:** the standalone `splitblocks` tool is deprecated and will be removed in a future release.
> Use [`mimirtool blocks split`](../../cmd/mimirtool) instead.

This program splits source blocks into new blocks where each spans at most a duration of time. Its documentation has moved to [`docs/internal/tools/splitblocks.md`](../../docs/internal/tools/splitblocks.md).
