# undelete-blocks

> [!WARNING]
> **Deprecated:** the standalone `undelete-blocks` tool is deprecated and will be removed in a future release.
> Use [`mimirtool blocks undelete`](../../cmd/mimirtool) instead.

This program is a disaster recovery tool that can restore deleted Mimir blocks in object storage. Its documentation has moved to [`docs/internal/tools/undelete-blocks.md`](../../docs/internal/tools/undelete-blocks.md).
