## Installation

## cortextool

```console
# download the binary (adapt os and arch as needed)
$ curl -fSL -o "cortextool" "https://github.com/grafana/cortex-tools/releases/download/v0.10.4/cortextool_0.10.4_linux_x86_64"

# make it executable
$ chmod a+x "cortextool"

# have fun :)
$ ./cortextool --help
```
